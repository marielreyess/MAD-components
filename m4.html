Shield controlador NFC/RFID Adafruit PN532 para Arduino + Extras

Descripción Hemos convertido nuestra popular placa de desarrollo Adafruit PN532 en un shield: la herramienta perfecta para cualquier aplicación RFID o NFC de 13,56 MHz. El shield NFC de Adafruit utiliza el chipset PN532 (el chip NFC más popular del mercado) y es el que viene integrado en prácticamente todos los teléfonos o dispositivos compatibles con NFC. Este chipset es muy potente y puede prácticamente todo, como leer y escribir en etiquetas y tarjetas, comunicarse con teléfonos (por ejemplo, para procesar pagos) y actuar como una etiqueta NFC. Si bien el controlador tiene muchas funciones, nuestra biblioteca Arduino actualmente solo admite la lectura y escritura de etiquetas, y no admite la comunicación entre teléfonos y shield, la emulación de etiquetas (que requiere un elemento seguro externo, solo disponible en NXP) ni otras funciones más avanzadas. NFC (Near Field Communications) permite la comunicación entre dos dispositivos muy cercanos. Es similar a un Bluetooth de muy corto alcance que no requiere autenticación. Es una extensión de RFID, así que todo lo que se puede hacer con RFID se puede hacer con NFC. Dado que puede leer y escribir etiquetas, siempre puede usarlo para proyectos con etiquetas RFID. Ofrecemos varias etiquetas que funcionan a la perfección con este chip. También es compatible con cualquier otra etiqueta NFC/RFID de tipo 1 a 4.

Información de Seguridad y Uso

El shield Adafruit fue diseñado por ingenieros de RF con el mejor equipo de prueba para crear un diseño y una antena con un alcance de 10 cm (4 pulgadas), el máximo alcance posible con la tecnología de 13,56 MHz. Puede fijarlo fácilmente detrás de una placa de plástico con separadores y leer tarjetas a través de una barrera (no metálica).
